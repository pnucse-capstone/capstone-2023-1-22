프로그램 실행후 menu에서 create 선택하시면 프로젝트명(임의), Code Prover의 Report, Bug Finder의 Report를 선택하는 창이 나옵니다.
각각의 Report는 함께 첨부한 test_project 압축파일 내의 각 프로젝트별 Result list를 선택하시면 됩니다. (_CP가 Code Prover, _BF가 Bug Finder에 해당)
선택을 완료하시고 잠깐 기다리면 완료되었다는 창과 함께 결과가 보여집니다.

Save를 선택하시면 현재 화면에 표시되어있는 결과를 프로젝트명.ser 이란 파일로 프로젝트 경로에 생성합니다.

Open을 선택하시면 save에서 저장한 파일을 선택하는 창이 나옵니다. 선택하시면 결과를 불러옵니다.

Help는 내부조작에 관한 내용입니다.
표에서 좌클릭을 한번 하면 셀을 선택할수 있고 좌클릭을 더블클릭하면 셀의 내용을 선택할 수 있습니다. 좌클릭으로 셀을 선택한 상태에서 우클릭을 더블클릭하면 셀의 내용을 별도 창으로 확인할 수 있습니다.
